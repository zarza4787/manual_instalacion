Pantalla Principal

En esta pantalla podemos observar que tenemos acceso a todas las ventanas de los ejercicios

Ejercicio 1 - Alta Nuevo Empleado

En este ventana, vamos a dar de alta un nuevo empleado en el sistema. Para ello, seguiremos los siguientes pasos:

  1. Introducimos el nombre del empleado
  2. Introducimos el apellido del empleado
  3. Introducimos el email del empleado
  4. Introducimos el numero de telefono del empleado
  5. Introducimos la fecha de alta del empleado
  6. Introducimos el departamento del empleado
  7. Seleccionamos el manager del empleado
  8. Una vez rellenados todos los campos, le damos a Guardar esto insertara el empleado en el sistema

Ejercicio 2 - Alta Nuevo Almacen

En esta ventana, vamos a dar de alta un nuevo almacen en el sistema. Para ello, seguiremos los siguentes pasos:

  1. Introducimos el nombre del almacen. Si no introducimos nombre el sistema no nos dejara guardar el almacen.
  2. Seleccionamos la region del almacen.
  3. Dependiendo de la region que seleccionemos, nos cargaran los paises de esa region
  4. En cuanto a la ubicaciones, puedo que algun pais no tengo ubicacion, si el pais tiene ubicaciones saldran. En caso de no selecionar una ubicacion se pondra este texto "No existen ubicaciones" y el sistema no dejara guardarlo e saltara un notificacion indicandolo.
  5. Una vez rellenados todos los campos, le damos a Guardar esto insertar el almacen en el sistema.

Ejercicio 3 - Modificar datos de un cliente

En esta ventana, vamos a poder modificar los datos de un cliente en el sistema. Para ello, seguiremos los siguientes pasos:

  1. Primero seleccionamos un cliente, en caso contrario nos saltara una notificacion diciendo "Por favor seleccione una fila"
  2. Una vez seleccionado el cliente, se nos abrira una ventana con los datos precargados en sus respetivos campos y podremos modificar sus datos en los campos correspondientes.
  3. Una vez modificados los datos, le damos a Guardar esto actualizara el cliente en el sistema.
